# YouTube Subscriptions List View Fix

## Description
This Chrome extension restores the classic **list view** layout for the YouTube subscriptions feed, replacing the modern forced grid view. It enforces a horizontal layout with fixed 16:9 thumbnails (300px width) and aligns metadata to the right for better readability and information density.

**Key Features:**
* Forces horizontal layout (Thumbnail | Text).
* Fixes thumbnail aspect ratio to strict 16:9 (prevents vertical/square cropping).
* Hides "Shorts" shelves and other clutter from the subscription feed.

## Installation
1.  **Download/Clone** this repository to a folder on your computer.
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked**.
5.  Select the folder containing the extension files (`manifest.json`, `styles.css`, `content.js`).
6.  Refresh your YouTube subscriptions page.
